# Changelog

## v0.5.0 (2026-04-26)

### Features
- 🖥️ Desktop GUI with Tauri (macOS, Linux, Windows)
- 🌐 Browser UI mode (`--browser`)
- 🔒 P2P encrypted sync via iroh/QUIC
- 📋 Clipboard history with unified timeline
- 🔄 Auto-update (CLI + GUI)
- 🤝 Pair codes for easy device setup
- 🌍 Hub/portal for device discovery
- 🔁 Relay transport for cross-network sync (Pro)
- 📦 One-liner install with pair code
- ⌨️ Global hotkey (Ctrl+B) for clipboard toggle
- 🖼️ Image, text, and file sync support
- 🔍 mDNS local network discovery
- 📊 2,400+ tests (Rust + Vitest + Playwright + Docker E2E)

### Platforms
- macOS (Intel + Apple Silicon)
- Linux (x64, headless, WSL)
- Windows (x64, NSIS installer)
